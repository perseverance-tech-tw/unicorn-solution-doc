Detailed Description

A collection of functions and variables for processing and displaying video frames.

This namespace encapsulates the logic for taking raw video frames, applying transformations (like remapping and resizing), and updating GTK image widgets asynchronously. It utilizes a thread pool to offload heavy processing from the main UI thread.

Processes a single frame asynchronously in a worker thread.

This is the core worker function. It applies the specified view transformation, handles alert border logic, and schedules the final UI update on the main GTK thread.

Parameters
fpA FunctionProcess struct containing all data for the task.

This is the core worker function. It applies the specified view transformation (Original, Panorama, Anypoint), handles the logic for displaying alert borders based on object detection results and zone policies, and finally queues the resulting image to be displayed on the main UI thread via g_idle_add.
