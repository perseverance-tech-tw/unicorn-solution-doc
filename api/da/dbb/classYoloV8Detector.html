Detailed Description

A class for performing object detection using a YOLOv8 model with the ONNX Runtime.

This class encapsulates the entire detection pipeline, including model loading, preprocessing, inference, and postprocessing (Non-Maximum Suppression).

◆ process_output()

void YoloV8Detector::process_output ( const std::vector< Ort::Value > &  output)

Processes the raw output tensors to extract bounding boxes, scores, and class IDs.

Postprocesses the raw model output to generate final detections.

This method applies confidence thresholding and Non-Maximum Suppression (NMS) to filter and refine the detection results, storing them in member variables.

The process involves several optimizations:

  1. Early Filtering: Iterates through all predictions (e.g., 8400), keeping only those whose highest class score exceeds the conf_threshold_.
  2. Top-K Selection: Sorts the filtered candidates by confidence and keeps only the top k (e.g., 300) to reduce the complexity of the subsequent NMS step.
  3. NMS: Applies Non-Maximum Suppression to eliminate overlapping boxes for the same object.
  4. Rescaling: Converts the final box coordinates from the model's input size (e.g., 640x640) back to the original image's dimensions.
    Parameters
    outputThe vector of output tensors from the inference method.
