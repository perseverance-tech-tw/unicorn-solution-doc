Navigates through the children of a GtkStack in a circular (carousel) manner.

This function determines the currently visible child of the stack, finds its position within the provided items array, and then switches the visible child to the next or previous item in the sequence. The navigation wraps around, creating a continuous loop.

Parameters
stackA pointer to the GtkStack widget that will be navigated.
itemsA NULL-terminated array of C-strings representing the names of the stack's children.
isPreviousA boolean that controls the navigation direction (true for previous, false for next).
