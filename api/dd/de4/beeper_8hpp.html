Detailed Description

Defines a header-only, thread-safe utility for playing audible alert sounds.

This file provides the Beeper namespace, which contains a set of inline functions for managing and playing different beep patterns (e.g., single burst, continuous loop) for VRU (Vulnerable Road User) alerts. It is designed to be self-initializing and uses system audio players like paplay or aplay.
