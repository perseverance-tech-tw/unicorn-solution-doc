Detailed Description

Main application class that handles GTK UI initialization, navigation, and signal connections.

The App class extends BaseView and manages the main GTK window, sidebar buttons, stack navigation, and signal connections. It serves as the entry point for coordinating the views, controllers, and global state of the application.

Initializes the main application window and user interface components. Loads layout from a Glade file, applies CSS styling, binds widgets, connects signals, and initializes cameras.

◆ ~App()

App::~App ( )

Destroys the App instance and releases GTK resources.

Destructor for App.

Releases the GtkBuilder reference using g_object_unref() to free all associated widgets loaded from the Glade file.

Note
Do not manually unreference individual widgets retrieved from the builder (such as window or stack) to avoid GLib-GObject-CRITICAL errors.
