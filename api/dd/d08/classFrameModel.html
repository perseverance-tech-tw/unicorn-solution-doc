Detailed Description

Manages a collection of FrameRecord objects.

This class provides an interface to create, read, update, and delete frame configurations. It also handles loading from and saving data to a SQLite database. It inherits from BaseModel to provide generic CRUD functionality.

◆ loadFromDatabase()

void FrameModel::loadFromDatabase ( )

Loads frame records from a SQLite database into memory.

It finds the first available database from Constants::DB_PATHS, clears the current in-memory records, and populates the model with data from the 'frames' table. If no database is found or is empty, it calls seeder().

◆ saveToDatabase()

void FrameModel::saveToDatabase ( )

Saves all in-memory frame records to the SQLite database(s).

This method iterates through all managed FrameRecord objects and persists them to the database paths defined in Constants::DB_PATHS. It will overwrite any existing data in the 'frames' table.
