Detailed Description

Defines the FrameProcessor namespace for handling video frame processing and display.

This header declares the structures, enums, global variables, and function signatures used for asynchronous frame processing. It defines the FunctionProcess struct to bundle all necessary data for a single processing task and outlines the public interface for the namespace.
