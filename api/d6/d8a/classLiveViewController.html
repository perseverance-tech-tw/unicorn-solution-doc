Detailed Description

Handles live camera viewing, playback, and UI interaction for real-time video processing.

The LiveViewController manages camera input, image/video playback, and user interactions such as switching between modes (Drive, Standby, Help), recording, and toggling AI detection. It integrates OpenCV for frame processing and GTK for GUI rendering.

This function sets up the view to display the original image. It hides anypoint frames, shows the panorama frame, updates the frame label, and sets the current view to ORIGINAL.
