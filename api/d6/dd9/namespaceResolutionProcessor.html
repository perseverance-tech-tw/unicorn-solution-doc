Detailed Description

Manages different resolution settings for remapping maps.

This namespace provides a system to define and switch between different map sizes (e.g., Native, Medium, Low). It dynamically calculates these sizes based on the camera's native parameters to provide different performance/quality trade-offs.

void ResolutionProcessor::initializeResolutions ( )

Dynamically calculates and populates the available resolution settings.

This function should be called once at startup. It retrieves the native map size from MoildevApplicator and calculates Medium and Low resolutions based on predefined scaling factors. It populates the global resolutionSetting map and sets the default to RES_NATIVE.
