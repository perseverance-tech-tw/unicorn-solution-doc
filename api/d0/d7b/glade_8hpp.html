Detailed Description

Declares a set of utility functions for working with GTK's Glade UI files and CSS.

This header provides a collection of helper functions to simplify common tasks such as loading Glade builder files, applying CSS for styling, managing dynamic button image effects (hover, active), and handling file paths.
