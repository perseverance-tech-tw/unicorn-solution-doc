Detailed Description

Handles playback operations including video management, deletion, and fullscreen control.

This controller is responsible for managing the Playback view, allowing users to:

  • Open saved recordings
  • Delete videos
  • Enter and exit fullscreen playback mode
  • Handle user interaction events in the GTK interface.
