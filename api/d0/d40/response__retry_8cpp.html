Displays a modal error dialog with "Retry" and "Close" options.

This function creates a GtkMessageDialog, adds custom "Retry" and "Close" buttons with their respective response IDs, and runs the dialog modally. It blocks execution until the user makes a selection. The function returns a boolean indicating whether the "Retry" option was chosen.

Parameters
parentA pointer to the parent GtkWindow.
messageThe error message to display inside the dialog.
Returns
bool Returns true if the user clicks "Retry", false otherwise.
