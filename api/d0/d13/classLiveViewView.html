Detailed Description

View for live view functionality, including media loading and display mode switching.

Manages the UI components and user interactions for the main live video display screen.

Provides UI component binding and connects signals to the controller via the Mediator.

The LiveViewView class is a central part of the UI, inheriting from BaseView. It is responsible for loading the live view layout from a Glade file, binding all its widgets (buttons, images, labels), and connecting them to appropriate handlers. It manages the state of different display modes and handles video playback controls for media files.

This method uses structured binding with vectors of pairs to efficiently retrieve and assign widget pointers from the GtkBuilder instance. It handles buttons, images, labels, event boxes, and other GTK widgets.
