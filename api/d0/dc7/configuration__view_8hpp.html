Detailed Description

Defines the ConfigurationView class for the application's configuration UI.

This header file declares the ConfigurationView class, which is responsible for managing the user interface of the camera configuration page. It includes member variables for all GTK widgets, static callback functions for handling user events, and methods for UI initialization and state management.
