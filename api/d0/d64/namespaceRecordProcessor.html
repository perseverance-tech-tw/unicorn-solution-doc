Detailed Description

Manages video recording functionality.

This namespace encapsulates the logic for creating, writing to, and saving video files from a stream of OpenCV frames (cv::Mat). It maintains the recording state internally.

◆ startRecord()

bool RecordProcessor::startRecord ( )

Starts the video recording process.

Creates a timestamped .mp4 file in the assets/recordings directory and prepares an OpenCV VideoWriter to write frames to it. The video properties (resolution, FPS) are derived from the global FrameProcessor::cap object.

Returns
bool Returns true if recording started successfully, false otherwise.

Stops the video recording process.
