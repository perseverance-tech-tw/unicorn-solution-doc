◆ getSizeFactorFromDatabase()

double Constants::getSizeFactorFromDatabase ( )

Retrieves the SIZE_FACTOR value from the database.

This function iterates through all paths defined in Constants::DB_PATHS and searches for a valid entry in the format_time table with the key size_factor. If a valid record is found, it returns the corresponding value. Otherwise, it falls back to the default (0.617).

Returns
The size factor (default 0.617 if not found).

Loads active camera parameters from the database.

Load active camera calibration from the database.
