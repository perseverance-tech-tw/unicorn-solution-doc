Detailed Description

A utility class for detecting and managing video cameras.

This class consists of static methods only and is not meant to be instantiated. It provides a cross-platform interface to find available cameras, get their names, and test their availability using OpenCV.

Detects all available video cameras connected to the system.

This is the main entry point for camera detection. It internally calls the appropriate platform-specific method (Windows, Linux, or macOS) to discover devices.

Returns

Checks if a specific camera index is accessible and can provide frames.

This method attempts to open a camera at the given index and read a single frame. It's a reliable way to verify that a device is not just listed but is also operational.
