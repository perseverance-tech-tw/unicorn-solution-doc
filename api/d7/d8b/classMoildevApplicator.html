Detailed Description

A singleton class that applies Moildev transformations to create and manage remapping maps.

This class acts as the main interface between the application's data models (like FunctionRecord) and the low-level Moildev library. Its responsibilities include initializing the Moildev engine with camera parameters from the database, generating base maps for different views (Anypoint, Panorama), applying modifications like cropping/margins, and regenerating all maps at different quality resolutions.

Applies cropping (margins) to a base map and saves the result.

It regenerates a clean, full-resolution base map, calculates a Region of Interest (ROI) based on the margin parameters in the functionRecord, crops the base map to this ROI, and then saves the final cropped map, overwriting the previous file.
