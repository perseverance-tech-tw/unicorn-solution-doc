Detailed Description

Implements the SettingController class, managing user interactions for the settings view.

This controller handles multiple configuration aspects, including:

  • Camera parameters (load, save, apply)
  • Resolution management
  • Datetime format preferences
  • Language and information pages

The controller communicates between GTK UI components and underlying models such as SQLite databases, ResolutionProcessor, and MoildevApplicator.
