Detailed Description

Defines the MoildevApplicator singleton class for high-level Moildev operations.

This header file declares the MoildevApplicator class, which serves as a high-level manager for the core Moildev library. It handles loading camera parameters, initializing the Moildev engine, and provides an interface to generate and manage remapping matrices (maps).
