Detailed Description

Implements the functions for the FrameProcessor namespace.

This file contains the implementation of the core logic for video frame processing. It includes functions for asynchronous task execution, image transformations (remapping, resizing), and updating GTK widgets safely from worker threads using g_idle_add.

GTK idle callback function to update a GtkImage widget.

This function is executed on the main GTK thread. It creates a GdkPixbuf from the provided pixel data and sets it on the target GtkImage. It is responsible for freeing the memory associated with IdleData and the pixel buffer.
