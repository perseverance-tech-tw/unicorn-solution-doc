Detailed Description

A static utility class for displaying simple GTK message dialogs.

This class is not meant to be instantiated. It provides a convenient, single-function interface to show modal dialog boxes to the user.

Creates, shows, and destroys a modal GTK message dialog.

This function simplifies the process of showing a message to the user. It creates a modal dialog with an OK button that blocks the parent window until it is closed.

Parameters
parentA pointer to the parent GtkWindow, which will be blocked by the dialog.
messageThe main text content to display inside the dialog.
titleThe text to display in the dialog's title bar.
typeThe MessageType (INFO, WARNING, ERROR) which determines the icon and style of the dialog.

This function maps the custom MessageType enum to the corresponding GTK message type, creates a new GtkMessageDialog, sets its title and message, and runs it modally. The dialog is automatically destroyed after the user interacts with it.
