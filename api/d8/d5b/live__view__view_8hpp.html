Detailed Description

Defines the LiveViewView class, which manages the user interface for the main live video display.

This header declares the LiveViewView class, responsible for all UI elements and interactions on the main 'Live' screen. It manages different display modes (e.g., Main, Panorama, MOIS, BSIS), handles video playback controls, and displays the processed video streams.
