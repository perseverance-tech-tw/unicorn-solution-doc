Detailed Description

Declares the Mediator class, which centralizes communication between controllers, views, and models.

The Mediator class implements the Singleton pattern to ensure a single shared instance that connects all MVC components in the application. It provides global access points for controllers, views, models, and utility classes (such as ThreadPool and YOLO detectors), effectively decoupling individual components while maintaining centralized coordination.
