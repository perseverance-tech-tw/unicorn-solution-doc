Detailed Description

A collection of functions to manage application-wide localization and translation.

This utility namespace handles loading language data from JSON files, translating text keys, applying translations to GTK widgets, managing persistent language settings, and handling text direction (LTR/RTL). It is designed to be used as a static utility.
