A lambda function that formats a double-precision floating-point number into a string.

This helper converts a double to a string with a specified precision. It then intelligently removes any unnecessary trailing zeros after the decimal point. If the number is a whole number after trimming (e.g., 123.), it ensures the format is "123.0" for clarity.

Parameters
valThe double value to be formatted.
precisionThe number of decimal places to use for the initial conversion. Defaults to 6.
Returns
std::string The formatted string representation of the double.

A lambda function that formats a double value, displaying it as an integer if it has no fractional part.
