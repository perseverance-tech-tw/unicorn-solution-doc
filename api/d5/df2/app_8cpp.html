Detailed Description

Implementation of the App class, which represents the main application window.

This file implements the App class responsible for initializing and managing the GTK-based user interface of the application. It loads the Glade layout, sets up widgets, connects signals, initializes cameras, and handles navigation between different views (Live, Configuration, Playback, Settings, Help).
