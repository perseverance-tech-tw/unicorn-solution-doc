Detailed Description

Defines the PlaybackView class for the application's video playback UI.

This header file declares the PlaybackView class, which manages the user interface for browsing and playing back recorded videos. It includes member variables for all GTK widgets, state variables for playback control, and static callback functions for handling user events.
