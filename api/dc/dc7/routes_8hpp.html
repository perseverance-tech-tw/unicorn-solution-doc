Detailed Description

Declares the setupRoutes function responsible for initializing and linking MVC components.

This header defines the setupRoutes() function, which connects views, controllers, and models through the Mediator pattern. It is called during application startup to initialize the GTK stack, controllers, and data models used across the app.

The function ensures that all UI views are properly bound, controllers are linked to their corresponding views, and models are registered in the Mediator for global accessibility.
