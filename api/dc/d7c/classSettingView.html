Detailed Description

Manages the user interface for the application's settings pages.

This class handles the UI logic for the settings screen, which is organized into multiple pages using a GtkStack. It is responsible for loading the UI from a Glade file, binding widgets, connecting user interaction signals to the SettingController, and updating the UI display.

void SettingView::populateResolutionOptions ( )

Dynamically creates and adds resolution radio buttons to the UI.

Iterates through the available resolution settings defined in ResolutionProcessor, creates a GtkRadioButton for each, groups them, and adds them to the resolutionRadioBox container. The currently active resolution is set as the default selected option.

void SettingView::viewBinding ( )
virtual

Binds UI widgets from the Glade file to class members.

Binds all UI widgets from the Glade file to class member variables.

Retrieves pointers to all necessary GTK widgets, such as stacks, buttons, labels, and entry boxes, using the GtkBuilder instance. It also programmatically manages the pages of the settings stack and sets initial widget states (e.g., making entry boxes non-editable).
