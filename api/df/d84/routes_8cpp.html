Detailed Description

Implements application route setup and component initialization.

This file defines the setupRoutes() function, responsible for initializing all core MVC components of the application — including detectors, models, views, and controllers — and registering them into the Mediator singleton. It effectively bootstraps the runtime environment by connecting data models, business logic, and user interface views for seamless interaction.

Sets up the application routes and component bindings.

Initializes and connects all major MVC components.

This function acts as the core initializer of the MVC structure. It:

  • Initializes AI detector(s) and registers them into the Mediator.
  • Instantiates FunctionModel and FrameModel and attempts to load existing database data.
  • Creates and binds all Views and Controllers (LiveView, Configuration, Playback, Setting).
  • Registers all components in the Mediator for centralized access.
  • Loads user settings and initializes the default Live View screen.
