Detailed Description

Singleton class that centralizes communication between MVC components.

The Mediator serves as a global communication hub between controllers, views, and models. It allows each component to access others without creating tight coupling, simplifying dependency management and improving modularity across the codebase.

Example usage:

Mediator::getInstance()->liveViewController->openPage();
Mediator::getInstance()->frameModel->updateFrame();
