Checks if a pointer is null and terminates the program on failure.

This function validates a given pointer. If it is nullptr, it prints a fatal error message to standard error and immediately terminates the application with a failure status using std::exit(). This is intended for critical pointers whose absence is considered an unrecoverable error during development.

Parameters
ptrThe const void* pointer to check.
msgAn optional custom message to display alongside the error if the pointer is null.
