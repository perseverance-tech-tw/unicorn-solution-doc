Detailed Description

Implementation of the Mediator class, which centralizes communication between controllers and views (MVC).

The Mediator class implements the Singleton pattern, providing a single coordination layer between controllers, models, and views. It ensures decoupling of components and acts as the central message bus that manages access to shared instances such as controllers, views, models, and global services (e.g., ThreadPool and object detectors).
