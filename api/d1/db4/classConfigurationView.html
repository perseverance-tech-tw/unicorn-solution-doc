Detailed Description

Manages the user interface for the camera configuration page.

This class loads the UI layout from a Glade file, binds all interactive widgets (buttons, spin controls, switches), and connects their signals to handler functions. It allows the user to modify camera parameters like alpha, beta, zoom, margins, and flip states for various preset functions (Anypoint, Panorama).
