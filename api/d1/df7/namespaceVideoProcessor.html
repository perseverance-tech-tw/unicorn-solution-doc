Detailed Description

Manages the high-level logic for video frame processing and view dispatching.

This namespace acts as the main coordinator for rendering video. Its primary function, dispatchCurrentViewJobs, determines the current application state (e.g., live view, playback, configuration), runs AI and risk assessments, and then creates and dispatches a set of rendering tasks to the FrameProcessor to be executed asynchronously.

◆ dispatchCurrentViewJobs()

void VideoProcessor::dispatchCurrentViewJobs ( )

The main dispatcher function that orchestrates frame processing.

This is the central function called in the main application loop. It determines the current context (live vs. playback), performs AI/risk evaluation, identifies the active view mode, and creates a list of rendering jobs (FunctionProcess) which are then sent to the FrameProcessor for asynchronous execution in a thread pool.
