Detailed Description

A collection of utilities for VRU (Vulnerable Road User) risk assessment.

This namespace provides a set of functions to prepare raw detection data, calculate Regions of Interest (ROIs), evaluate risks for MOIS and BSIS systems based on object positions within these zones, and generate visual overlays for debugging and display.

Prepares a detection packet and ROI overlays.

This function is the main entry point. It calculates scaled ROIs, retrieves a detection snapshot via the callback, and prepares two output frames: one for further geometric transformation and one for displaying the original view with overlays.

Parameters

aiEnabledA flag indicating whether AI detection is active.

Returns
