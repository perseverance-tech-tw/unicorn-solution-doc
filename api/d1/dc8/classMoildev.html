Detailed Description

Core class for handling fisheye image transformations and de-warping.

This class encapsulates a specific fisheye camera's calibration parameters. It provides methods to generate cv::remap matrices (mapX, mapY) for various projections like Anypoint (rectified) and Panorama. It also includes utilities for coordinate conversions between the fisheye image plane and angular (alpha, beta) space, along with performance-oriented approximation functions.
