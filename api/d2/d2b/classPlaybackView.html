Detailed Description

Manages the user interface for the video playback page.

This class is responsible for the UI where users can select a recorded video from a list and watch it. It handles the video playback controls (play, pause, seek), manages different display modes (Original, Panorama, MOIS, BSIS), and interacts with the PlaybackController to perform its functions.

◆ onVideoPlaybackTimeout()

gboolean PlaybackView::onVideoPlaybackTimeout ( gpointer  userData)
static

Glib timeout callback for video playback, called for each frame.

Glib timeout callback for video playback.

This is the core function of the video player. It is called repeatedly by the GTK main loop. It reads a frame, updates the UI (progress bar and time labels), assigns the frame to the FrameProcessor for rendering, and saves the frame to lastFrame for use when pausing.

Parameters
userDataA pointer to the PlaybackView instance.
Returns
gboolean G_SOURCE_CONTINUE to keep the timer running, G_SOURCE_REMOVE to stop it.
