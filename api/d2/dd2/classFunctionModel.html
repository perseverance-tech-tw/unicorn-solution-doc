Detailed Description

Manages a collection of FunctionRecord objects.

This class provides an interface to create, read, update, and delete function presets. It handles loading data from and saving data to a SQLite database, as well as managing the serialization of map files. It is implemented as a singleton.

void FunctionModel::loadMap ( FunctionRecord &  record)

Loads the mapX and mapY matrices for a record from its file paths.

If map files do not exist, it attempts to generate them. If loading or generation fails, the record's map data is cleared.
